# Requisitos del sistema

El sistema AnimeFull debe cumplir con varias funciones principales para que pueda trabajar como un sistema experto para una tienda de anime.

Primero, el sistema debe permitir registrar productos con datos como nombre, anime, categoria, precio, stock y stock minimo.

Tambien debe permitir consultar el inventario para saber que productos estan disponibles, cuales tienen poco stock y cuales estan agotados.

El sistema debe poder registrar clientes de forma sencilla, para despues relacionarlos con pedidos o recomendaciones.

Otra funcion importante sera generar pedidos o ventas, donde se pueda seleccionar un producto, cantidad y total.

Tambien debe tener recomendaciones, para que el usuario pueda escribir lo que busca y el sistema trate de sugerir un producto que coincida con su presupuesto y gustos.

El sistema debe mostrar alertas cuando un producto tenga bajo stock o este agotado.

Tambien debe aplicar reglas de inferencia para tomar decisiones, por ejemplo detectar bajo stock, recomendar reabastecimiento o buscar productos dentro del presupuesto.

Por ultimo, el sistema debe explicar algunas decisiones, para que el usuario entienda por que se recomendo cierto producto o por que se genero una alerta.
