# HERRAMIENTAS QUE SE EMPLERAN DURANTE EL PROYECTO

En este proyecto de SE, el cual se llamara AnimeFULL, se empleran diferentes herramientas, que ayudaran a crear la interfaz, la base de datos y la logica del sistema experto.

python sera la herramienta principal, ya que en este se programaran, los agentes, las reglas de inferencia y las funciones principales del SE.

Streamlit lo utilizare para crear la parte visual y estetica del proyecto la idea principal que tengo es que se vea , como un sistema de tienda departamental, donde se veran los datos, imagenes, botones, tablas, colores, entre otros.

SQlite lo utilizare como la base de datos local, en el cual se encarga de guardar, los productos, clientes, pedidos, ventas y alertas de inventario.

Pandas sera para llevar un orden o manejo de informacion en forma de tablas, como un inventario o historial de pedidos.

Plotly sera para agregar como graficas de los datos de la tienda, como ventas, productos mas vendidos, menos vendidos, solo por dar ejemplo.

Github sera el principal, para guardar toda la informacion del proyecto, como avances, commits.
y por ultimo el VS code sera el programa del SE, donde se desarrollara y organizara el codigo del proyecto.
