# Interfaz visual

La interfaz visual de AnimeFull sera una parte importante del proyecto, porque quiero que no se vea como un programa simple, sino como un sistema de tienda mas completo.

La idea es que tenga un diseño parecido a un sistema de punto de venta o tienda departamental, con botones, colores, tablas, tarjetas de productos, imagenes y alertas.

Para mejorar el diseño visual, primero se realizara un prototipo en Figma. Esto servira como guia para definir mejor el menu, las pantallas, los colores, las tarjetas de productos, las alertas y la forma en que se vera el sistema.

Despues ese diseño se pasara a Streamlit, donde ya se programara la interfaz funcional del proyecto.

El sistema tendra secciones como inventario, pedidos, clientes, recomendaciones, alertas de bajo stock y agentes del sistema.

En el inventario se podran ver los productos registrados con su precio, categoria, anime, stock y estado.

En la parte de recomendaciones, el usuario podra escribir lo que busca, por ejemplo algo de Naruto con cierto presupuesto, y el sistema tratara de recomendar un producto.

Tambien se mostraran alertas cuando un producto tenga poco stock o este agotado.

El objetivo de esta interfaz es que el proyecto sea facil de usar, se vea mas presentable y parezca un sistema real de tienda.
