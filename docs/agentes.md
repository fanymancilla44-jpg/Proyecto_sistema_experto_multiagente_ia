# Agentes del sistema

En el sistema AnimeFull se utilizaran diferentes agentes para que cada parte tenga una funcion especifica y no este todo mezclado.

El agente cliente se encargara de recibir lo que el usuario escriba, por ejemplo si busca algo de Naruto, One Piece o algun producto con cierto presupuesto.

El agente inventario revisara los productos registrados, el precio, la categoria y el stock disponible.

El agente recomendador buscara productos que coincidan con lo que quiere el cliente, tomando en cuenta anime, presupuesto y disponibilidad.

El agente pedidos se encargara de registrar pedidos o ventas dentro del sistema.

El agente reabastecimiento revisara que productos tienen bajo stock o estan agotados, para sugerir que se compren mas unidades.

El agente explicador dira por que el sistema recomendo cierto producto o por que marco algun producto como bajo stock.

La idea de usar agentes es que el sistema se vea mas organizado y pueda funcionar como un sistema experto, ya que cada agente analiza una parte diferente del problema.
